Introduction to Pandas

Pandas is a powerful Python library for data manipulation and analysis. It provides high-performance, easy-to-use data structures and tools for working with structured data.

Whether you're a data scientist, analyst, or developer, Pandas is an essential tool for:

  • Data cleaning and preparation
  • Data analysis and exploration
  • Time series analysis
  • Statistical operations
  • Data visualization

Installation and Setup

Installing Pandas is straightforward using pip:

pip install pandas

To get started, import Pandas in your Python script:

import pandas as pd
import numpy as np  # Often used alongside Pandas

Core Data Structures

Series

A Series is a one-dimensional labeled array that can hold data of any type:

# Creating a Series
s = pd.Series([1, 3, 5, np.nan, 6, 8])
print(s)

# Output:
# 0    1.0
# 1    3.0
# 2    5.0
# 3    NaN
# 4    6.0
# 5    8.0
# dtype: float64

DataFrame

A DataFrame is a 2-dimensional labeled data structure with columns of potentially different types:

# Creating a DataFrame
df = pd.DataFrame({
    'Name': ['John', 'Anna', 'Peter'],
    'Age': [28, 22, 35],
    'City': ['New York', 'Paris', 'London']
})
print(df)

# Output:
#     Name  Age      City
# 0   John   28  New York
# 1   Anna   22     Paris
# 2  Peter   35    London

Basic Operations

Reading Data

Pandas can read data from various file formats:

# Reading CSV
df = pd.read_csv('data.csv')

# Reading Excel
df = pd.read_excel('data.xlsx')

# Reading JSON
df = pd.read_json('data.json')

Data Selection

Accessing data in a DataFrame:

# Select a single column
names = df['Name']

# Select multiple columns
subset = df[['Name', 'Age']]

# Filter rows
adults = df[df['Age'] >= 18]

# Select by position
first_row = df.iloc[0]

# Select by label
row_by_name = df.loc[0, 'Name']

Data Analysis Examples

Let's look at some common data analysis operations:

Basic Statistics

# Summary statistics
print(df.describe())

# Mean age
print(df['Age'].mean())

# Count by city
print(df['City'].value_counts())

Data Grouping

# Group by city and calculate mean age
city_stats = df.groupby('City')['Age'].mean()
print(city_stats)

Pro Tip

When working with large datasets, use the df.info() method to get a quick overview of your data, including memory usage and data types.
